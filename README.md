# AI-Powered Career Intelligence Platform

## Project Overview

The AI-Powered Career Intelligence Platform is an end-to-end data analytics project designed to analyze job market trends, salary insights, hiring locations, and in-demand technical skills for analytics careers.

This project combines Python, NLP, PostgreSQL, SQL, and Power BI to transform raw job posting data into actionable career intelligence.

---

## Business Problem

Job seekers often struggle to understand:

* Which skills are most in demand
* Which locations offer the most opportunities
* Expected salary ranges
* Career paths for Data Analyst, Business Analyst, Data Scientist, and Data Engineer roles

This platform provides data-driven insights to support career planning.

## Project Workflow

### 1. Data Collection

Collected and imported job posting data.

### 2. Data Cleaning

* Removed duplicates
* Handled missing values
* Filtered analytics-related roles
* Created salary metrics

### 3. NLP Skill Extraction

Extracted technical skills from job descriptions:

* SQL
* Excel
* Python
* Power BI
* Tableau
* AWS
* Azure
* Spark
* Snowflake
* ETL
* Statistics
* Machine Learning
* Data Visualization

### 4. PostgreSQL Database

Created analytics database and imported cleaned job data.

### 5. SQL Business Analysis

Performed business-focused queries:

* Top hiring roles
* Top hiring locations
* Salary analysis
* Remote opportunity analysis
* Job type analysis

### 6. Power BI Dashboards

#### Executive Career Intelligence Dashboard

Provides job market overview and hiring trends.

#### Skills Intelligence Dashboard

Analyzes demand for technical skills.

#### Salary Intelligence Dashboard

Explores salary trends across roles and locations.

#### Career Recommendation Engine

Provides career guidance based on role demand, salary potential, and required skills.

---

## Dashboard Highlights

* 1,612 analytics job postings analyzed
* 957 unique hiring companies
* Average salary approximately $89.85K
* 29.5% remote opportunities
* SQL identified as the most in-demand skill

---

## Key Insights

* Business Analyst and Data Analyst roles show strong market demand.
* SQL, Excel, and Python remain the most valuable analytics skills.
* Data Engineering roles offer the highest salary potential.
* Remote opportunities continue to represent a significant share of analytics jobs.
